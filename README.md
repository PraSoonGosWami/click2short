## Click2Short
### About the app
Click2Short is an Url shortening service like bitly and tinyurl.

Click2Short is made with ReactJs, NodeJs, Express and MongoDb (the MERN Stack)

You can create your account, create customised short url which can be easily rememberd and shared.

Click2Short also provides you with the analytics which helps your track number of hits and your user engagement
